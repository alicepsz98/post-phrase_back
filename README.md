## :rocket: About the Project

### Back-end system of a social network, PostPhrase, with basic implementations for user and post:
  - login
  - creation (user / post)
  - get all (user / post)
  - get by id (user / post)
  - edit (user / post)
  - delete (user / post)

---

## 👩‍💻 Technologies

- Node.js
- Express
- Knex
- Bcrypt
- JWT
- UUID
- Dotenv
- SQLite
